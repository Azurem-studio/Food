About the Cook

Hi, I'm Azurem! This site started as a way to keep track of recipes I didn't want to lose, and it slowly turned into a full-blown hobby. I'm not a professional chef — just someone who really likes good food and enjoys figuring out how to make it at home.

When I'm not in the kitchen, I'm usually writing, worldbuilding, or falling down some other creative rabbit hole. Cooking ended up being the perfect break from all that screen time.

Site Notes

Four extras added for the final project, beyond the earlier course assignments:

  • Live recipe search — a JavaScript filter on the Recipes page that hides non-matching cards as you type, with a dynamically shown "no results" message.
  • Image carousel — the Featured Dishes carousel on the Home page. Navigation is manual only (prev/next buttons and dot indicators) — autoplay is intentionally off.
  • Sticky footer — a flexbox layout (body as a column flex container, main set to flex: 1) keeps the footer pinned to the bottom of the viewport on every page, even short ones, regardless of browser window height.
  • Google Maps plugin — the embed above. I customized the query to a specific address and set z=15 for a closer zoom than the default, then wrapped it in a responsive container matching the site's existing video embeds.
