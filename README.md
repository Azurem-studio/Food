## Final Project Notes

**Responsive design:** the site has a distinct desktop layout, tablet layout, and mobile layout (not
just resized/recolored — the recipe grid, card shape, and column spans all change between
breakpoints).

**Four enhancements added beyond the earlier course assignments:**

1. **Live recipe search** (Recipes page) — a JavaScript filter that hides non-matching recipe
   cards as you type, with a dynamically shown "no results" message.
2. **Image carousel** (Home page, "Featured Dishes") — manual prev/next buttons and dot
   indicators only; autoplay is intentionally disabled.
3. **Sticky footer** — a flexbox layout keeps the footer pinned to the bottom of the browser
   window on every page, even short ones, at any window height.
4. **Google Maps plugin** (About page) — an embedded map of the Ann Arbor Farmers Market. The
   default embed query and zoom level were both customized (`z=15`), and it's wrapped in a
   responsive container matching the site's existing video embeds.

**Validation:** run all three pages through the W3C Markup Validator and WAVE before submitting.
